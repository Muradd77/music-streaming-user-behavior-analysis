# 🎧 Music Streaming User Behavior Analysis

## 📌 Introduction
With the rise of digital music platforms, understanding user listening behavior has become essential for improving recommendations and user experience. This project explores user interaction data from a music streaming platform using data analysis techniques.

---

## 🎯 Objective
The main goals of this project are:

- Analyze user listening behavior
- Identify patterns in music consumption
- Explore relationships between user activity and preferences
- Generate insights for recommendation systems

---

## 📊 Dataset
The dataset contains information about user activity on a music streaming platform.

Features include:

- User listening habits
- Track-related attributes
- Time-based activity
- Behavioral indicators

> Note: The dataset has been preprocessed before analysis.

---

## 🔍 Exploratory Data Analysis (EDA)

EDA steps performed:

- Data distribution analysis
- User behavior trends
- Correlation analysis
- Pattern identification
- Visualization using:
  - Matplotlib
  - Seaborn

Key insights:

- Users show distinct listening patterns based on time/activity
- Some features strongly correlate with user engagement
- Behavioral clusters can be identified
